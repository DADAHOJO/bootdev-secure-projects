# Boot.dev Secure Projects

This repository contains security-focused portfolio projects built while progressing through Boot.dev.

## Goal

The purpose of this repo is to convert Boot.dev software engineering concepts into practical cybersecurity, software security, and DevSecOps projects.

## Positioning

Boot.dev provides the programming, backend, and DevOps foundation. I use OWASP Top 10, OWASP ASVS, and NIST SSDF to add a security engineering layer.
